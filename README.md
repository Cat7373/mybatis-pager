# mybatis-pager-spring-boot-starter
为 SpringBoot(基于 Servlet 的 Web 项目) + Mybatis 提供简单易用的分页查询支持

## 环境要求
* `Java8`及以上
* `SpringBoot2.x` + 基于`Servlet`的`Web`支持
* `Mybatis 3.x`
